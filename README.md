# Задание 9.1 Создание конвертора цветов

## Описание 

Программа позволяет конвертировать цвета из HEX в RGB. Демонстрируя цвет на background. Компонент ColorConverter используется в App.tsx

__Функции__

- `convertToRgb` - конвертирует HEX в RGB и возвращает значения `R`, `G`, `B` и `alpha` и присваивает `setRgbColor` состояние.
- `checkHexType` - проверяет HEX на валидность и возвращает `true` или `false`.
- `handleChangeColorValue` - обрабатывает изменение значения HEX в инпуте.

Дополнительно использую useEffect чтобы при изменении colorValue менялся background.
Состояние colorValue и rgbColor используется для хранения значений HEX и RGB.

Добавил доп. функцию при двойном клике поле импута сбрасывает его в исходное состояние.

## Зависимости 

Приложение использует библиотеку [hex-rgb](https://www.npmjs.com/package/hex-rgb) для конвертирования HEX в RGB. Остальной код реализован самостоятельно.
